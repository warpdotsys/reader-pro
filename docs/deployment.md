# Deployment

## Docker image

Image: `warpdotsys/reader-pro`

Recommended tags:

- `3.2.14`
- `v3.2.14`
- `latest` from the default branch
- short Git SHA tags generated by GitHub Actions

## Runtime configuration

The default port is `8080`. The default working directory inside the container is `/data`.

Common environment variables/JVM properties:

- `READER_APP_WORKDIR=/data`
- `READER_SERVER_PORT=8080`
- `READER_APP_SECURE=false`
- `READER_APP_SECUREKEY=`
- `READER_APP_USERLIMIT=15`

Mount `/data` as a persistent volume.

## Repository secrets

GitHub Actions expects these repository secrets:

- `DOCKERHUB_USERNAME`
- `DOCKERHUB_TOKEN`

Use a DockerHub access token instead of an account password when possible. Do not commit credentials or pass them inline in commands.
